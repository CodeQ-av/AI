# Code Q#

## 🧠 AI Models Repository  
This repository contains a collection of machine learning models and scripts, organized into Jupyter notebooks (`.ipynb`). Below is an overview of the included files:  

### 📂 **Models**  
- **DecisionTreeClassifier.ipynb** – Implements a Decision Tree Classifier to predict outcomes based on structured data.  
- **DeepCNN.ipynb** – A Deep Convolutional Neural Network (CNN) for image classification tasks.  
- **FeedForwardNetwork.ipynb** – A simple feedforward neural network for general classification problems.  
- **HelloAIWorld.ipynb** – A beginner-friendly notebook demonstrating basic AI concepts.  
- **KNearestNeighbors.ipynb** – K-Nearest Neighbors algorithm for classification tasks.  
- **LinearRegressionPredictor.ipynb** – Linear Regression model to predict continuous outcomes.  
- **LogisticRegressionPredictor.ipynb** – Logistic Regression model for binary classification.  
- **NaviesBayesClassifier.ipynb** – Naive Bayes classifier for probabilistic classification.  
- **micrograd.ipynb** – A minimalistic autograd engine for building neural networks from scratch.  

### 🛠️ **Other Files**  
- **.gitattributes** – Configuration file for defining attributes per path.  
- **desktop.ini** – System file for folder settings (can be ignored).  

### 🚀 **Usage**  
1. Clone the repository:  
```bash
git clone https://github.com/CodeQ-av/AI.git
```
2. Open Jupyter Notebook:  
```bash
jupyter notebook
```
3. Navigate to the notebook and run the code.
